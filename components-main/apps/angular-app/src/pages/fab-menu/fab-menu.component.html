FAB Menu

A Floating Action Button that expands into a menu of related actions. Replaces the traditional speed dial with more expressive motion.

Interactive Demo

Click the FAB to toggle the menu. Notice the shape morphing and rotation animation.

♿ Accessibility

  • Keyboard Support: The FAB is a button that can be activated with Enter/Space.
  • Focus Management: When opened, focus should be managed to the menu items (implementation dependent).
  • ARIA: The FAB has aria-label="Menu" by default.
