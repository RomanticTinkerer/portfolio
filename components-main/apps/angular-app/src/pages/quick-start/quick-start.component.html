Quick Start

Integrate Material 3 Web Components into any project in minutes.

CDN — no build step

Import directly in your HTML via jsDelivr. Works in any modern browser with no tooling required.

Install via package manager

Install individual packages from npm for use in framework projects.

npm

pnpm

Angular

Add CUSTOM_ELEMENTS_SCHEMA to your component or module to prevent template errors for unknown tags.

React

React 19+ natively supports Custom Elements. Listen to custom events by prefixing with on — e.g. onbutton-click.

Vue

Vue has native support for Custom Elements. Use standard Vue directives like @button-click to listen to custom events.
