Positioning API

Use placement to choose where the menu opens relative to its anchor, and offset to control the gap in pixels.

Placement Options

Built into m3-menu

Bottom

bottom-start bottom-center bottom-end

Top

top-start top-center top-end

Right

right-start right-center right-end

Left

left-start left-center left-end

Example: placement="right-center" offset="12"

♿ Accessibility

  • Role Mapping: The surface uses role="menu" and items use role="menuitem".
  • Keyboard Navigation: Arrow keys, Home, End, Escape, and Tab dismissal are supported.
  • Dismissal: Menus close on outside click, Escape, Tab, or item selection.
